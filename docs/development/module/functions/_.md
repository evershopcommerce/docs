# _

Template translation function for React components with variable interpolation.

## Import

```typescript
import { _ } from '@evershop/evershop/lib/locale/translate/_;
```

## Syntax

```typescript
_(text: string, values?: Record<string, string>): string
```

### Parameters

**`text`**

**Type:** `string`

The text to display, with optional placeholders using `${variable}` syntax.

**`values`**

**Type:** `Record<string, string>` (optional)

Object containing values to replace placeholders.

## Return Value

Returns `string` with placeholders replaced by values.

## Examples

### Basic Usage

```typescript
import { _ } from '@evershop/evershop/lib/locale/translate/_;

// Simple text
const message = _('Welcome to our store');

// With variables
const greeting = _('Hello ${name}', { name: 'John' });
// Returns: "Hello John"
```

### In React Components

```typescript
import React from 'react';
import { _ } from '@evershop/evershop/lib/locale/translate/_;

export default function ProductCard({ product }) {
  return (
    <div>
      <h2>{_(product.name)}</h2>
      <p>{_('Price: ${price}', { price: product.price })}</p>
      <p>{_('${qty} items in stock', { qty: product.qty })}</p>
    </div>
  );
}
```

### Multiple Variables

```typescript
import { _ } from '@evershop/evershop/lib/locale/translate/_;

const message = _(
  'Order ${orderId} for ${total} has been ${status}',
  {
    orderId: '12345',
    total: '$99.99',
    status: 'completed'
  }
);
// Returns: "Order 12345 for $99.99 has been completed"
```

### Conditional Messages

```typescript
import { _ } from '@evershop/evershop/lib/locale/translate/_;

const itemCount = 5;
const message = _(
  '${count} items in cart',
  { count: itemCount.toString() }
);
```

## Notes

- Used primarily in React components and templates
- Placeholders use `${variable}` syntax
- If a placeholder key is not found in values, the placeholder is left unchanged
- Returns the original text if no values provided
- To use server-side translations with CSV support, use the `translate` function instead
